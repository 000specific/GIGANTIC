Secretome Filter Manifest Builder

Pick the biological features each protein must satisfy. Each enabled feature becomes one filter. A protein joins the secretome only if it passes every enabled filter (ANDed).
 
Output name is appended to the auto-incremented counter. Blank → just secretome_NNN. With "moroz_strict" → secretome_NNN_moroz_strict.
Each card below is one biological feature. Tick the checkbox to include it in the filter. Within each feature, add one or more clauses across one or more tools, and choose how they combine (ALL = AND, ANY = OR).
